# Student Mental Health

This repository contains a Django-based application focused on student mental health assistance.

## Project Structure

- `mental/`: Django project and application code.
- `ml_training/`: Data files used for machine learning training.

## Setup

Refer to `SETUP_GUIDE.md` for instructions on installing dependencies and running the project locally.

## Usage

Start the development server:

```bash
python manage.py runserver
```

Navigate to `http://localhost:8000` to access the application.
